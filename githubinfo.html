All About Github

  • Q. What is a version control system?
  • A. A version control system is the practice of tracking and managing changes to software code.
  • Q. What is Git?
  • A. Git is a version control system that is used for source code management.
  • Q. What is Github?
  • A. Github is a code hosting platform for version control and collaberation where one or many people can work on a project individually or together.
  • Q. What is the difference between Git and Github?
  • A. Git is a version control system that lets you manage and track your source code history, whereas Github is a cloud based hosting service that lets your manage Git repositories.
  • Q. Who started Github and how was it started?
  • A. Chris Wanstrath, Tom Preston-Werner, P.J. Hyett, and Scott Chacon started Github.
  • Q. What company owns it now?
  • A. Microsoft currently owns Github.
  • Q. How mush does a Github account cost?
  • A. A Github account doesn't cost any money because it is free.
  • Q. What is the octocat?
  • A. The octocat is the official mascot for Github (part cat, part octopus).
